# GIF quine

This is a quine (a program that outputs itself) coded in C that outputs a GIF.

I'm using the [Spleen](https://github.com/fcambus/spleen) font.

## The code / The output

![output](./output.gif)

## Usage

```sh
gcc quine.c -o quine
./quine > output.gif
```
